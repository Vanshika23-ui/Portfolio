A data structure is a unique way of storing or organizing the data in computer memory so that it can be later used effectively.Data Structures is classified in Linear data structures and non-linear data structures.Personally , i belive that data structures in java is way more interesting.It seems a little hard in the start but trust me it is the most interesting subject i've learned.

Linear Data Structures :

In a linear data structure all the elements are arranged in the linear or sequential order.It is also called as single level data structure.

Non-Linear Data Structures:

The non-linear data structure does not arrange the data in a sequential manner. Non-linear data structures are also known as multilevel data structure.

Types of Data Structure in Java :

Arrays
Linked Lists
Stack
Queue
Graph
Set

Arrays : Arrays can have data items of simple and similar types such as int or float, or even user-defined datatypes like structures and objects.The syntax of the array is int arr[] or int [] arr. Also to create new array the syntax is int arr[] = new arr[].

Linked List : A Linked List is a collection of similar types of data elements, called nodes, which point to the next following nodes by means of pointers.

Stack :A stack is a LIFO (Last In First Out) data structure that can be physically implemented as an array or as a linked list. Insertion and deletion of elements in a stack occur at the top end only. An insertion in a stack is called pushing and deletion from a stack is called popping.For example there are the number of plates arranged in the kitchen so the plate which was last inserted will be out first as the stack definition says.

Queue: A queue is a FIFO (First In First Out) data structure and we can physically implement it either as an array or a linked list. Whatever way we use to implement a queue, insertions always take place at the “rear” end and deletions always from the “front” end of the queue.In queue we can take the example of sales.

Graph : A set of a finite number of vertices which we call as nodes.An edge with a finite set of ordered pairs which is in the form.

Set : A Set is a special data structure in which we can not use the duplicate values. It is a very useful data structure mainly when we want to store unique elements, for example - unique IDs.
